# claude-code-skills

A collection of Claude Code skills.

## mystatus

Morning dashboard showing Linear tickets and GitHub PRs scoped by organization. Run `/mystatus mai` or `/mystatus gm` to see what needs attention without visiting multiple UIs.

## tmux-title

Automatically sets your tmux tab title to describe what Claude Code is working on. Claude calls this proactively at the start of tasks so you can see at a glance what each session is doing.
